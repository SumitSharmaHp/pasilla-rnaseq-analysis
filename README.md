# Pasilla RNA-seq Analysis

End-to-end RNA-seq differential expression analysis using FastQC, fastp, HISAT2, SAMtools, featureCounts, and DESeq2.

## Dataset
Drosophila melanogaster Pasilla RNA-seq dataset

## Results
- Genes analyzed after filtering: 9868
- Upregulated genes: 364
- Downregulated genes: 404
- Significant genes: 768
